Primeiro projeto

Esse é o meu primeiro projeto FrontEnd utilizando HTML e CSS. Esse projeto foi inspirado no Desafio 10 do Curso em Vídeo, mas que possui um toque totalmente personalizado! Além disso, alguns sites me auxiliaram no aprendizado dessas duas tecnologias: a W3C School e a MDN Web Docs. Recomendo muito.

O projeto em si é bastante modesto, mas estou orgulhosa por tê-lo feito. É o início de um grande sonho: uma realização pessoal, um trabalho feito com muito carinho e dedicação. Todas as lições passadas pelo professor Gustavo Guanabaraserviram para construir cada detalhe desse projeto.

Imagem responsiva

Durante meus estudos, um dos primeiros conceitos de responsividade do qual tive conhecimento foi com as imagens. No início, adaptá-las em qualquer resolução não foi uma tarefa fácil, uma vez que não dominava as propriedades de altura (height) e largura (width). Mas agora esse processo se tornou bastante simples.

No exemplo abaixo, eu criei algumas imagens coloridas com resoluções diferentes e fui modifcando os valores com base na largura do browser (navegador). A extensão Window Resizer me auxiliou a definir os valores das imagens conforme o tamanho do display de cada dispositivo.

Imagem com efeito Parallax

Esse foi um dos recursos apresentados pelo professor Gustavo Guanabara sobre o efeito Parallex. O efeito consiste em manter a imagem fixa enquanto o cursor do mouse se movimenta.

É uma técnica simples que trás uma dinâmica na imagem bastante interessante!
